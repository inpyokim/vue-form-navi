> a vue directive for arrow key navigation

## Install

```sh
npm i vue-form-navi
```

```ts
// globally
import formNavi from 'vue-form-navi';

Vue.directive('form-navi', formNavi);

// component option...
import formNavi from 'vue-form-navi';


{
  ...
  directives: { formNavi },
  ...
}
```
